# Bone-To-Peak 게임 기획 총괄

> 최종 수정일: 2026-02-28

---

## 게임 콘셉트

**Bone-To-Peak**은 다크 판타지 Bullet Heaven (Survivors-like) 게임이다. 플레이어는 네크로맨서로서 직접 싸우는 대신, 적의 시체를 자원으로 활용하여 언데드 군단을 지휘한다. 단순한 자동 공격이 아닌 **미니언 관리**와 **전술적 소환**이 핵심이다.

### 기본 정보

| 항목   | 내용                                              |
| ------ | ------------------------------------------------- |
| 장르   | Action Roguelike / Bullet Heaven / Horde Survivor |
| 테마   | Dark Necromancy / Tactical Legion Building        |
| 플랫폼 | Steam (PC)                                        |
| 엔진   | Unity 6.3 LTS (2D URP)                            |

## 차별점 (기존 Survivors-like 대비)

### 1. 직접 전투 → 군단 지휘

Vampire Survivors 등 기존 작품은 플레이어가 직접 공격하고 회피하는 구조다. Bone-To-Peak에서 네크로맨서는 **지휘관**이다. 플레이어의 전투력은 미니언 군단의 규모와 구성에 달려 있다.

### 2. 적 = 자원

기존 게임에서 적은 경험치만 주는 장애물이다. Bone-To-Peak에서 적의 시체는 **소환 · 합성 · 폭발**의 핵심 자원이다. 적을 처치할수록 군단이 기하급수적으로 성장한다.

### 3. 전술적 포메이션

자동 발사체 대신 미니언의 **배치와 진형**이 전투를 결정한다. 상황에 맞는 포메이션 전환이 생존의 핵심이다.

### 4. 지수적 성장 곡선

"Bone"(뼈 하나)에서 시작하여 "Peak"(정점)에 도달하는 극적인 성장을 체감할 수 있다. 초반의 단일 스켈레톤에서 후반의 수백 언데드 군단까지, 시각적으로도 명확한 파워 곡선을 제공한다.

### 5. 엔드리스 도전

Wave 20 클리어 후 끝나지 않는다. **엔드리스 모드**에서 무한 스케일링되는 적과 맞서며 최고 기록에 도전할 수 있다. 리더보드와 결합하여 반복 플레이 동기를 부여한다.
